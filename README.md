# fitness-timer-autopost
Автопостинг релевантных фото+текстов в Telegram и на сайт TabataTimer.

## Основное
- Telegram: `https://t.me/s/fitnesstimer`
- Блог: `https://www.tabatatimer.ru/blog.html`
- Библиотека контента: `content_library.json`

## Документация
- `DOCS_AUTPOSTING.md` — основная краткая инструкция
- `AUTOPOST_SETUP.md` — настройка автопостинга
- `ACTUAL_CONFIG.md` — актуальные переменные окружения
- `YANDEX_CLOUD_SETUP.md` — настройка хранения на Yandex Cloud

## Запуск
Workflow запускается по расписанию или вручную через GitHub Actions UI.
